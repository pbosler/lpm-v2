Detailed Description

Data structure and methods for remapping spherical LPM data using SSRFPACK.
.

Uses the interfaces provided by ssrfpackInterface module. For references describing the SSRFPACK package and its counterpart STRIPACK, see the ssrfpackInterface module's detailed description.

Performs a remesh/remap of an LPM SphereBVE simulation using indirect interpolation for all variables in a BVE mesh. Remaps to reference time t = t_{rm}, where t_{rm} is time of definition for a reference mesh (numerical solution). Note that the reference mesh's Lagrangian parameter has been reset, so that x = x0, y = y0, z = z0 at t = t_{rm}.

Parameters
[in,out]newSpheretarget SphereBVE mesh (note that this must have been allocated prior to calling this subroutine)
[in]oldSpheresource SphereBVE mesh
[in]refSphereSphereBVE reference mesh
[in]refRemeshremesh data structure associated with refSphere
[in]AMR.TRUE. if adaptive refinement will be used
[in]flagFn1FlagFunction for vorticity refinement, must have same interface as refinementmodule::FlagFunction
[in]tol1tolerance for flagFn1
[in]desc1description of AMR criterion used for flagFn1
[in]flagFn2FlagFunction for vorticity refinement, must have same interface as refinementmodule::FlagFunction
[in]tol2tolerance for flagFn2
[in]desc2description of AMR criterion used for flagFn2
[in]RefineFLowMapYNTrue if refinement of the flow map will be used
[in]flowMapVarToltolerance value for Lagrangian coordinate variation per face
