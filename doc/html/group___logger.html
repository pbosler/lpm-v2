Logger module

A Logger object for writing output to console or to files. More...

Detailed Description

A Logger object for writing output to console or to files.

Handles display, formatting, and organization of console messages. All messages are presumed to have a key - value pair. For example, the key may be the origin of the message in the code, and the value contains the message content.
