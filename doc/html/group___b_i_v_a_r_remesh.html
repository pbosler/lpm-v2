Detailed Description

Data structure and methods for remapping planar LPM data using the BIVAR package. Uses the interfaces provided by BIVARInterface.

For references describing the BIVAR package, see the BIVARInterface module's detailed description.

Performs a remesh/remap of an LPM simulation using indirect interpolation of all variables in a PlanarIncompressible mesh. Remaps to reference time t = t_{rm}, where t_{rm} is time of definition for a reference mesh (numerical solution). Note that the reference mesh's Lagrangian parameter has been reset, so that x = x0, y = y0 at t = t_{rm}.
