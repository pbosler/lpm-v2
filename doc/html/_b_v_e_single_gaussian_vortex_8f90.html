Detailed Description

Driver program for the problem of a single Gaussian vortex on a rotating sphere.

Author
Peter Bosler, Sandia National Laboratories Center for Computing Research

Driver program for the problem of a single Gaussian vortex on a rotating sphere.
Demonstrates the solution of an inviscid incompressible flow on the sphere using SphereBVE, SphereBVESolver, Refinement, and SSRFPACKRemesh.

Late-time vorticity distribution of an initially Gaussian vortex on a rotating sphere (coarse resolution).

real(kreal) function bvesinglegaussianvortexdriver::setgaussconst ( type(bvemesh), intent(in)  aBVEMesh)

Defines a constant to ensure that the total integral of vorticity is zero over the whole sphere.

Parameters
[in]aBVEMeshSphereBVE mesh
Returns
C such that

\[ \int_S \zeta(\vec{x})\,dA = \int_{S} \zeta_G(\vec{x})\,dA - C = 0, \]

where \( \zeta_G \) is a Gaussian distribution on a sphere.
