461 C Given data values and gradients at the three vertices of

463 C computes the value of F at P where F interpolates the ver-
464 C tex data. Along the triangle sides, the interpolatory
465 C function F is the Hermite interpolatory tension spline
466 C defined by the values and tangential gradient components
467 C at the endpoints, and the gradient component normal to the
468 C triangle side varies linearly with respect to arc-length
469 C between the normal gradient components at the endpoints.
470 C A first-order C-1 blending method is used on the underly-
471 C ing planar triangle. Since values and gradients on an arc
472 C depend only on the vertex data, the method results in C-1
473 C continuity when used to interpolate over a triangulation.
474 C
475 C The blending method consists of taking F(P) to be a
476 C weighted sum of the values at PP of three univariate Her-
477 C mite interpolatory tension splines defined on the line
478 C segments which join the vertices to the opposite sides and
479 C pass through PP: the central projection of P onto the
480 C underlying planar triangle. The tension factors for these
481 C splines are obtained by linear interpolation between the
482 C pair of tension factors associated with the triangle sides
483 C which join at the appropriate vertex.
484 C
485 C A tension factor SIGMA associated with a Hermite interp-
486 C olatory tension spline is a nonnegative parameter which
487 C determines the curviness of the spline. SIGMA = 0 results
488 C in a cubic spline, and the spline approaches the linear
489 C interpolant as SIGMA increases.

510 C
511 C Input parameters are not altered by this function.
512 C
513 C On output:
514 C

516 C

601  u2n = u2n + u2(i)*u2(i)

603  1 CONTINUE

605 C Compute Q1, Q2, and Q3.
606 C
607  u1n = sqrt(u1n)
608  u2n = sqrt(u2n)
609  u3n = sqrt(u3n)
610  DO 2 i = 1,3
611  q1(i) = u1(i)/u1n
612  q2(i) = u2(i)/u2n
613  q3(i) = u3(i)/u3n
614  2 CONTINUE

616 C Compute interpolated value (VAL) at P by looping on
617 C triangle sides.
618 C
619  val = 0.
620 C
621 C Contribution from side opposite V1:
622 C
623 C Compute value and gradient at Q1 by interpolating
624 C between V2 and V3.
625 C
626  CALL arcint(q1,v2,v3,f2,f3,g2,g3,sig1, f,g,dum)

628 C Add in the contribution.
629 C

631  ds = -(g(1)*v1(1) + g(2)*v1(2) + g(3)*v1(3))/u1n
632  sig = (b2*sig3 + b3*sig2)/s1

634 C
635 C Contribution from side opposite V2:
636 C
637 C Compute value and gradient at Q2 by interpolating
638 C between V3 and V1.
639 C
640  CALL arcint(q2,v3,v1,f3,f1,g3,g1,sig2, f,g,dum)
641 C
642 C Add in the contribution.
643 C
644  dv = g2(1)*u2(1) + g2(2)*u2(2) + g2(3)*u2(3)
645  ds = -(g(1)*v2(1) + g(2)*v2(2) + g(3)*v2(3))/u2n
646  sig = (b3*sig1 + b1*sig3)/s2
647  val = val + c2*hval(b2,f2,f,dv,ds,sig)
648 C
649 C Contribution from side opposite V3:
650 C
651 C Compute interpolated value and gradient at Q3
652 C by interpolating between V1 and V2.
653 C
654  CALL arcint(q3,v1,v2,f1,f2,g1,g2,sig3, f,g,dum)
655 C
656 C Add in the final contribution.
657 C
658  dv = g3(1)*u3(1) + g3(2)*u3(2) + g3(3)*u3(3)
659  ds = -(g(1)*v3(1) + g(2)*v3(2) + g(3)*v3(3))/u3n
660  sig = (b1*sig2 + b2*sig1)/s3
661  fval = val + c3*hval(b3,f3,f,dv,ds,sig)
662  RETURN
663  END

679 C Given a triangulation of a set of nodes on the unit
680 C sphere, along with data values H and gradients GRAD at the
681 C nodes, this subroutine determines, for each triangulation

683 C that the Hermite interpolatory tension spline H(A), de-
684 C fined by SIGMA and the endpoint values and directional
685 C derivatives, preserves local shape properties of the data.
686 C In order to define the shape properties on an arc, it is
687 C convenient to map the arc to an interval (A1,A2). Then,
688 C denoting the endpoint data values by H1,H2 and the deriva-
689 C tives (tangential gradient components) by HP1,HP2, and
690 C letting S = (H2-H1)/(A2-A1), the data properties are
691 C

693 C nonpositive,
694 C and
695 C
696 C Convexity: HP1 .LE. S .LE. HP2 or HP1 .GE. S

1084  f0 = fneg
1085  go to 5

1088 C

1092  lp2 = lstptr(lend(n2),n1,list,lptr)
1093  sigma(lp2) = sig

1104 C

1110 C

1113  RETURN
1114 C

1116 C
1117  12 dsmax = dsm
1118  ier = -2

1238 C Given a triangulation of N nodes on the unit sphere with
1239 C data values F at the nodes and tension factors SIGMA asso-
1240 C ciated with the arcs, this routine uses a global method
1241 C to compute estimated gradients at the nodes. The method
1242 C consists of minimizing a quadratic functional Q(G) over
1243 C the N-vector G of gradients, where Q approximates the
1244 C linearized curvature of the restriction to arcs of the
1245 C interpolatory function F defined by Function FVAL. The
1246 C restriction of F to an arc of the triangulation is the
1247 C Hermite interpolatory tension spline defined by the data
1248 C values and tangential gradient components at the endpoints
1249 C of the arc. Letting D1F(A) and D2F(A) denote first and
1250 C second derivatives of F with respect to a parameter A var-
1251 C ying along a triangulation arc, Q is the sum of integrals
1252 C over the arcs of D2F(A)**2 + ((SIGMA/L)*(D1F(A)-S))**2,
1253 C where L denotes arc-length, SIGMA is the appropriate ten-
1254 C sion factor, and S is the slope of the linear function of
1255 C A which interpolates the values of F at the endpoints of
1256 C the arc.
1257 C
1258 C Since the gradient at node K lies in the plane tangent
1259 C to the sphere surface at K, it is effectively defined by
1260 C only two components -- its X and Y components in the coor-
1261 C dinate system obtained by rotating K to the north pole.
1262 C Thus, the minimization problem corresponds to an order-2N
1263 C symmetric positive-definite sparse linear system which is
1264 C solved by a block Gauss-Seidel method with 2 by 2 blocks.
1265 C
1266 C An alternative method, Subroutine GRADL, computes a
1267 C local approximation to the gradient at a single node and,
1268 C although less efficient when all gradients are needed, was
1269 C found to be generally more accurate (in the case of uni-
1270 C form zero tension) when the nodal distribution is very
1271 C dense, varies greatly, or does not cover the sphere.
1272 C GRADG, on the other hand, was found to be slightly more

4076  SUBROUTINE smsgs (N,X,Y,Z,U,LIST,LPTR,LEND,IFLGS,
4077  . sigma,w,p, nit,dfmax,f,grad, ier)
4078  INTEGER N, LIST(*), LPTR(*), LEND(n), IFLGS, NIT, IER

4080  . dfmax, f(n), grad(3,n)
4081 C
4082 C***********************************************************
4083 C
4084 C From SSRFPACK
4085 C Robert J. Renka
4086 C Dept. of Computer Science
4087 C Univ. of North Texas
4088 C renka@cs.unt.edu
4089 C 07/25/96
4090 C
4091 C This subroutine solves the symmetric positive definite
4092 C linear system associated with minimizing the quadratic
4093 C functional Q(F,FX,FY,FZ) described in Subroutine SMSURF.
4094 C Since the gradient at node K lies in the plane tangent to
4095 C the sphere surface at K, it is effectively defined by only
4096 C two components -- its X and Y components in the coordinate
4097 C system obtained by rotating K to the north pole. Thus,
4098 C the minimization problem corresponds to an order-3N system
4099 C which is solved by the block Gauss-Seidel method with 3 by
4100 C 3 blocks.
4101 C
4102 C On input:
4103 C
4104 C N,X,Y,Z,U,LIST,LPTR,LEND,IFLGS,SIGMA,W = Parameters
4105 C as described in Subroutine SMSURF.
4106 C
4107 C P = Positive smoothing parameter defining Q.
4108 C
4109 C The above parameters are not altered by this routine.
4110 C
4111 C NIT = Maximum number of iterations to be used. This
4112 C maximum will likely be achieved if DFMAX is
4113 C smaller than the machine precision. NIT .GE.
4114 C 0.
4115 C
4116 C DFMAX = Nonnegative convergence criterion. The
4117 C method is terminated when the maximum
4118 C change in a solution F-component between
4119 C iterations is at most DFMAX. The change in
4120 C a component is taken to be the absolute
4121 C difference relative to 1 plus the old value.
4122 C
4123 C F = Initial estimate of the first N solution compo-
4124 C nents.
4125 C
4126 C GRAD = 3 by N array containing initial estimates of
4127 C the last 3N solution components (the gradi-
4128 C ent with FX, FY, and FZ in rows 1, 2, and 3,
4129 C respectively).
4130 C
4131 C On output:
4132 C
4133 C NIT = Number of Gauss-Seidel iterations employed.
4134 C
4135 C DFMAX = Maximum relative change in a solution F-
4136 C component at the last iteration.
4137 C
4138 C F = First N solution components -- function values
4139 C at the nodes.
4140 C
4141 C GRAD = Last 3N solution components -- gradients at
4142 C the nodes.
4143 C
4144 C IER = Error indicator:
4145 C IER = 0 if no errors were encountered and the
4146 C convergence criterion was achieved.
4147 C IER = 1 if no errors were encountered but con-
4148 C vergence was not achieved within NIT
4149 C iterations.
4150 C IER = -1 if N, P, NIT, or DFMAX is outside its
4151 C valid range on input. F and GRAD are
4152 C not altered in this case.
4153 C IER = -2 if all nodes are collinear or the
4154 C triangulation is invalid.
4155 C IER = -3 if duplicate nodes were encountered.
4156 C
4157 C SSRFPACK modules required by SMSGS: APLYRT, CONSTR,
4158 C GRCOEF, SNHCSH
4159 C
4160 C Intrinsic functions called by SMSGS: ABS, ATAN, MAX, SQRT
4161 C
4162 C***********************************************************
4163 C
4164  INTEGER IFL, ITER, ITMAX, J, K, LPJ, LPL, NN
4165  REAL(kind=8) ALFA, ALFSQ, C11, C12, C13, C22, C23, C33,

4167  . dfmx, dgk(3), dgx, dgy, fk, g1, g2, g3, gjk,
4168  . gkj, pp, r1, r2, r3, rr2, rr3, sig, sinal, sx,
4169  . sy, t, t1, t2, t3, t4, t5, t6, tol, xj, xk,
4170  . xs, yj, yk, ys, zj, zk

4174  pp = p
4175  itmax = nit
4176  tol = dfmax

4178 C Test for errors in input and initialize iteration count,
4179 C tension factor, and output value of DFMAX.
4180 C
4181  IF (nn .LT. 3 .OR. pp .LE. 0. .OR. itmax .LT. 0
4182  . .OR. tol .LT. 0.) go to 5
4183  iter = 0
4184  sig = sigma(1)
4185  dfmx = 0.

4187 C Top of iteration loop.
4188 C
4189  1 IF (iter .EQ. itmax) go to 4

4192 C Loop on nodes.
4193 C
4194  DO 3 k = 1,nn
4195  xk = x(k)
4196  yk = y(k)
4197  zk = z(k)

4199  g1 = grad(1,k)
4200  g2 = grad(2,k)
4201  g3 = grad(3,k)
4202 C
4203 C Construct the rotation mapping node K to the north pole.
4204 C
4205  CALL constr(xk,yk,zk, cx,sx,cy,sy)
4206 C
4207 C Initialize components of the order-3 system for the
4208 C change (DF,DGX,DGY) in the K-th solution components.
4209 C

4212  c13 = 0.

4270  r2 = r2 + t*xj
4271  r3 = r3 + t*yj
4272 C
4273 C Bottom of loop on neighbors.
4274 C
4275  IF (lpj .NE. lpl) go to 2
4276 C

4278 C

4281  cc33 = c11*c33 - c13*c13

4286  . c11 .EQ. 0.) go to 6

4289  df = (r1 - c12*dgx - c13*dgy)/c11
4290 C

4292 C system, and update GRAD( ,K), F(K), and DFMX.
4293 C
4294  CALL aplyrt(dgx,dgy,cx,sx,cy,sy, dgk)
4295  grad(1,k) = g1 + dgk(1)
4296  grad(2,k) = g2 + dgk(2)
4297  grad(3,k) = g3 + dgk(3)
4298  f(k) = fk + df
4299  dfmx = max(dfmx,abs(df)/(1.+abs(fk)))
4300  3 CONTINUE
4301 C
4302 C Increment ITER and test for convergence.
4303 C
4304  iter = iter + 1
4305  IF (dfmx .GT. tol) go to 1
4306 C
4307 C The method converged.
4308 C
4309  nit = iter
4310  dfmax = dfmx
4311  ier = 0
4312  RETURN
4313 C
4314 C The method failed to converge within NIT iterations.
4315 C
4316  4 dfmax = dfmx
4317  ier = 1
4318  RETURN
4319 C
4320 C Invalid input parameter.
4321 C
4322  5 nit = 0
4323  dfmax = 0.
4324  ier = -1
4325  RETURN
4326 C
4327 C Node K and its neighbors are collinear.
4328 C
4329  6 nit = 0
4330  dfmax = dfmx
4331  ier = -2
4332  RETURN
4333 C
4334 C Nodes J and K coincide.
4335 C
4336  7 nit = 0
4337  dfmax = dfmx
4338  ier = -3
4339  RETURN
4340  END
4341  SUBROUTINE smsurf (N,X,Y,Z,U,LIST,LPTR,LEND,IFLGS,
4342  . sigma,w,sm,smtol,gstol,lprnt, f,
4343  . grad,ier)
4344  INTEGER N, LIST(*), LPTR(*), LEND(n), IFLGS, LPRNT,
4345  . ier
4346  REAL(kind=8) X(n), Y(n), Z(n), U(n), SIGMA(*), W(n), SM,
4347  . smtol, gstol, f(n), grad(3,n)
4348 C
4349 C***********************************************************
4350 C
4351 C From SSRFPACK
4352 C Robert J. Renka
4353 C Dept. of Computer Science
4354 C Univ. of North Texas
4355 C renka@cs.unt.edu
4356 C 07/21/98
4357 C
4358 C Given a triangulation of N nodes on the unit sphere with
4359 C data values U at the nodes and tension factors SIGMA
4360 C associated with the arcs, this routine determines a set of
4361 C nodal function values F and gradients GRAD = (FX,FY,FZ)
4362 C such that a quadratic functional Q1(F,GRAD) is minimized
4363 C subject to the constraint Q2(F) .LE. SM for Q2(F) =
4364 C (U-F)**T*W*(U-F), where W is a diagonal matrix of positive
4365 C weights. The functional Q1 is an approximation to the
4366 C linearized curvature over the triangulation of a C-1 fun-
4367 C ction F(V), V a unit vector, which interpolates the nodal
4368 C values and gradients. Subroutines INTRC1 and UNIF may be
4369 C called to evaluate F at arbitrary points.
4370 C
4371 C The smoothing procedure is an extension of the method
4372 C for cubic spline smoothing due to C. Reinsch -- Numer.
4373 C Math., 10 (1967) and 16 (1971). Refer to Function FVAL
4374 C for a further description of the interpolant F. Letting
4375 C D1F(T) and D2F(T) denote first and second derivatives of F
4376 C with respect to a parameter T varying along a triangula-
4377 C tion arc, Q1 is the sum of integrals over the arcs of
4378 C D2F(T)**2 + ((SIGMA/L)*(D1F(T)-S))**2 where L denotes arc-
4379 C length, SIGMA is the appropriate tension factor, and S is
4380 C the slope of the linear function of T which interpolates
4381 C the values of F at the endpoints of the arc. Introducing
4382 C a smoothing parameter P, and assuming the constraint is
4383 C active, the problem is equivalent to minimizing Q(P,F,
4384 C GRAD) = Q1(F,GRAD) + P*(Q2(F)-SM). The secant method is
4385 C used to find a zero of G(P) = 1/SQRT(Q2) - 1/SQRT(SM)
4386 C where F(P) satisfies the order-3N symmetric positive def-
4387 C inite linear system obtained by setting the gradient of Q
4388 C (treated as a function of F and GRAD with GRAD tangent to
4389 C the sphere surface) to zero. The linear system is solved
4390 C by the block Gauss-Seidel method (refer to SMSGS).
4391 C
4392 C Note that the method can also be used to select grad-
4393 C ients for the interpolation problem (F = U, SM = 0, and P
4394 C infinite). This is achieved by a call to Subroutine
4395 C GRADG.
4396 C
4397 C On input:
4398 C
4399 C N = Number of nodes in the triangulation. N .GE. 3.
4400 C
4401 C X,Y,Z = Arrays of length N containing Cartesian
4402 C coordinates of the nodes.
4403 C
4404 C U = Array of length N containing data values at the
4405 C nodes.
4406 C
4407 C LIST,LPTR,LEND = Data structure defining the trian-
4408 C gulation. Refer to STRIPACK
4409 C Subroutine TRMESH.
4410 C
4411 C IFLGS = Tension factor option:
4412 C IFLGS .LE. 0 if a single uniform tension
4413 C factor is to be used.
4414 C IFLGS .GE. 1 if variable tension is desired.
4415 C
4416 C SIGMA = Uniform tension factor (IFLGS .LE. 0), or
4417 C array containing tension factors associated
4418 C with arcs in one-to-one correspondence with
4419 C LIST entries (IFLGS .GE. 1). Refer to Sub-
4420 C programs GETSIG, SIG0, SIG1, and SIG2.
4421 C
4422 C W = Array of length N containing positive weights
4423 C associated with the data values. The recommend-
4424 C ed value of W(I) is 1/DU**2 where DU is the
4425 C standard deviation associated with U(I). DU**2
4426 C is the expected value of the squared error in
4427 C the measurement of U(I). (The mean error is
4428 C assumed to be zero.)
4429 C
4430 C SM = Positive parameter specifying an upper bound on
4431 C Q2(F). Note that F is constant (and Q2(F)
4432 C is minimized) if SM is sufficiently large that
4433 C the constraint is not active. It is recommend-
4434 C ed that SM satisfy N-SQRT(2N) .LE. SM .LE. N+
4435 C SQRT(2N).
4436 C
4437 C SMTOL = Parameter in the open interval (0,1) speci-
4438 C fying the relative error allowed in satisfy-
4439 C ing the constraint -- the constraint is
4440 C assumed to be satisfied if SM*(1-SMTOL) .LE.
4441 C Q2 .LE. SM*(1+SMTOL). A reasonable value
4442 C for SMTOL is SQRT(2/N).
4443 C
4444 C GSTOL = Nonnegative tolerance defining the conver-
4445 C gence criterion for the Gauss-Seidel method.
4446 C Refer to parameter DFMAX in Subroutine
4447 C SMSGS. A recommended value is .05*DU**2,
4448 C where DU is an average standard deviation
4449 C in the data values.
4450 C
4451 C LPRNT = Logical unit on which diagnostic messages
4452 C are printed, or negative integer specifying
4453 C no diagnostics. For each secant iteration,
4454 C the following values are printed: P, G(P),
4455 C NIT, DFMAX, and DP, where NIT denotes the
4456 C number of Gauss-Seidel iterations used in
4457 C the computation of G, DFMAX denotes the max-
4458 C imum relative change in a solution component
4459 C in the last Gauss-Seidel iteration, and DP
4460 C is the change in P computed by linear inter-
4461 C polation between the current point (P,G) and
4462 C a previous point.
4463 C
4464 C Input parameters are not altered by this routine.
4465 C
4466 C On output:
4467 C
4468 C F = Array of length N containing nodal function val-
4469 C ues unless IER < 0.
4470 C
4471 C GRAD = 3 by N array whose columns contain gradients
4472 C of F at the nodes unless IER < 0.
4473 C
4474 C IER = Error indicator and information flag:
4475 C IER = 0 if no errors were encountered and the
4476 C constraint is active -- Q2(F) is ap-
4477 C proximately equal to SM.
4478 C IER = 1 if no errors were encountered but the
