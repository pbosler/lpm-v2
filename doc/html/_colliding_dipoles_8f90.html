Driver program for the problem of two colliding Lamb dipoles in the plane.

Author
Peter Bosler, Sandia National Laboratories Center for Computing Research

Driver program for the problem of two colliding Lamb dipoles in the plane.
Demonstrates the solution of an inviscid incompressible flow in the plane using PlanarIncompressible, PlanarIncompressibleSolver, Refinement, and BIVARRemesh.

Initial vorticity distribution for colliding dipoles problem.

real *8 function bessj (   N,
real *8  X 
)

This function calculates the first kind modified Bessel function of integer order N, for any REAL X.

We use here the classical recursion formula, when X > N. For X < N, the Miller's algorithm is used to avoid overflows.

REFERENCE: C.W.CLENSHAW, CHEBYSHEV SERIES FOR MATHEMATICAL FUNCTIONS, MATHEMATICAL TABLES, VOL.5, 1962.

Parameters
[in]norder of Bessel function
[in]x
Returns
Bessel function output, \( B_n(x) \)
